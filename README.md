# Gubongo
Csodálatos FéregTv - chatjét vizualizáló web alkalmazás. Egy websocket szolgáltatáshoz csatlakozva megjeleníti a csatorna chatjére író felhasználók "kukacát".

# Install
```sh
npm install
```

# Build
```sh
npm run build
```
Open dist/index.html

# Development
```sh
npm run start
```
Open http://localhost:8080
